# Profile Builder Skill

## 推断维度
- 行业领域（互联网/SaaS/金融/教育/...）
- 职能角色（产品/运营/创业者/投资/管理/...）
- 所处阶段（早期探索/成长期/成熟期/...）
- 近期关注点（最近2周高频出现的主题）
- 知识积累深度（深/中/浅，按领域）

## 推断逻辑
- 看内容的主题分布，而非单篇文章
- 近期内容权重高于早期内容
- 操作型内容多 → 执行者角色
- 论点型内容多 → 思考者/决策者角色
- 数据型内容多 → 分析型角色

## 输出格式（写入 PROFILE.md）
```markdown
# 认知画像
更新时间：YYYY-MM-DD

## 行业领域
[推断结果]

## 职能角色
[推断结果]

## 所处阶段
[推断结果，基于近期内容推断]

## 近期关注点
- [主题1]
- [主题2]

## 知识积累深度
- 深：[领域]
- 中：[领域]
- 浅：[领域]
```
