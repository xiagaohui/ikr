# Source Finder Skill

## 使用时机
当用户问题涉及需要权威来源支撑时，在回答末尾附加延伸参考。

## 延伸参考来源类型
- 学术研究（论文、研究报告）
- 行业权威报告（A16Z、麦肯锡、IDC 等）
- 经典著作（书名 + 相关章节）
- 原始案例来源（公司官方博客、创始人访谈）

## 呈现格式
```
延伸参考（源头资料）：
• [标题] - [来源机构/作者]
• [标题] - [来源机构/作者]
```

## 注意
- 延伸参考独立于回答主体，不混入方案建议
- 不确定来源真实性时，不列出
- 最多列 2-3 条，精不求多
